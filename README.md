# SV_GRS-Polentina Nyapu Integration
Repositorio de integración controlada para aplicar portraits estilo Nyapu sobre un snapshot limpio de SVVE.
## Carpetas
- Mods/: snapshot limpio funcional del modpack base.
- nyapu/: carpeta de insumos descargados de Nyapu/AhCuteArt/etc.
- docs/: reportes técnicos de Codex.
## Reglas
- No tocar la carpeta real de Steam desde Codex.
- No instalar XNB.
- Preferir Content Patcher sobre overwrites.
- No usar overwrite si existe CP equivalente.
- Instalar por fases.
- Validar SMAPI después de cada fase.
- Documentar todo cambio.
